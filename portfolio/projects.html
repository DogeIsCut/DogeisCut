      • A Game about a Box

        A Game about a Box is a physics based 2D platformer where you take control of a blue box named Boxen. In the game's story, you must return Boxen to 'her place' after being seperated from the rest of the group. Throughout the game you encounter various unique gameplay mechanics, such as crumbling blocks, or turning into various things.

        Development

        A Game about a Box is developed in the Godot game engine, specifically, Godot 4.1. The game actually started development in Godot 3.5 but was later upgraded post release of Godot 4.0/

        The idea of the game sprouted randomly, much like many of my other projects. This currently remains my most completed game project to date, if you don't count the Scratch projects.
