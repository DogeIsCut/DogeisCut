      • A Game about a Box

        A Game about a Box is a physics based 2D platformer where you take control of a blue box named Boxen. In the game's story, you must return Boxen to 'her place' after being seperated from the rest of the group. Throughout the game you encounter various unique gameplay mechanics, such as crumbling blocks, or turning into various things.

        Development

        A Game about a Box is developed in the Godot game engine.
